# 央企公务用车管理系统 - 需求规格说明书 (V1)

Feature Name: official-vehicle-management
Updated: 2026-06-24
Version: V1（第一版基础功能）

## 引言

本系统面向中央企业公务用车使用管理场景，实现车辆全生命周期管理、用车审批流程数字化、费用基础管控、合规监管透明化。系统覆盖从车辆登记、用车申请、审批调度、派车出车、行程归库到费用记录的基础业务闭环。

V1 聚焦核心业务流程，复杂子场景、智能引擎、深度分析和增强功能预留至 V2。

## 术语表

| 术语 | 定义 |
|------|------|
| 公务用车 | 央企为保障公务活动配备的机动车辆 |
| 用车申请单 | 用车人提交的包含出行目的、时间、人数等信息的申请 |
| 派车单 | 调度员在审批通过后分配的车辆与司机信息单据 |
| 调度 | 车队管理员根据申请需求分配车辆和驾驶员的过程 |
| 出车记录 | 车辆从出库到归库的完整行程记录 |
| GPS轨迹 | 车辆行驶过程中采集的定位坐标序列 |
| 归库 | 车辆使用完毕后返回指定停放地点 |
| 三龄两证一检 | 驾驶员管理核心资质体系：驾龄/工龄/年龄，驾驶证/从业资格证，年度体检 |
| 准驾车型 | 驾驶证核定的可驾驶车辆类型（A1/A2/B1/B2/C1/C2） |
| 编制 | 部门公务用车核定的配备数量上限 |

## V1 与 V2 功能边界

### V1 已实现（本次交付）

核心业务流程主线 + 基础辅助功能：

- 车辆基础档案管理，支持编制限额管控
- 年检/保险到期日期记录与提醒
- 用车申请提交（6种基本场景），基本合规校验
- 两级审批流程（部门负责人→分管领导），通过/驳回
- 基础调度派车（手动选择车辆和驾驶员）
- 出车确认、归库登记、用车人确认
- GPS位置记录、轨迹回放、里程统计
- 费用记录、审核和基础统计
- 仪表盘核心指标展示和基础报表导出
- 8种固化角色权限控制
- 驾驶员基础档案、驾驶证/资格证管理、培训计划执行与逾期管控、体检记录、违章记录、状态自动管理
- 车辆运营管理五大场景：加油记录/维修保养/保险台账/年检记录/违章处理

### V2 预留（后续版本）

- 30+子场景细分类别和智能填表辅助
- 智能匹配引擎（车辆规格/拼车/调度资源匹配）
- 信用评分体系
- 审批加签/转审/代理/时效催办
- 甘特图时间线可视化
- 紧急插队调度
- 电子围栏、实时监控、轨迹比对
- 费用预算、油耗考核、ETC对账
- 日报/月报/大屏驾驶舱
- 自定义角色与权限模板
- 消息模板与多渠道触达
- 保密管理、合同管理、驾驶员画像
